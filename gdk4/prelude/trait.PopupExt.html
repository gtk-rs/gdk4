fn present(&self, width: i32, height: i32, layout: &PopupLayout) -> bool

Present self after having processed the PopupLayout rules. If the popup was previously now showing, it will be showed, otherwise it will change position according to layout.

After calling this function, the result of the layout can be queried using Popup::get_position_x, Popup::get_position_y, Surface::get_width, Surface::get_height, Popup::get_rect_anchor and Popup::get_surface_anchor.

Presenting may have fail, for example if it was immediately hidden if the self was set to autohide.

width

the unconstrained popup width to layout

height

the unconstrained popup height to layout

layout

the PopupLayout object used to layout

Returns

false if it failed to be presented, otherwise true.
